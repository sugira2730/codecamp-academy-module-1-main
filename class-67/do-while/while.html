JavaScript Loops: While & Do...While

Understanding conditional loops in JavaScript

What is a Loop?

A loop allows a program to execute a block of code repeatedly while a specific condition remains true. Loops are essential when we need to perform repetitive tasks without writing the same code multiple times.

In this lesson we will study two conditional loops:

  • while loop
  • do...while loop

The While Loop

The while loop executes a block of code repeatedly as long as a given condition evaluates to true.

The condition is checked before each iteration. If the condition is false from the beginning, the loop will never run.

Syntax

                
                    while (condition) {
                        // code to execute
                    }

Example

                    
                        let counter = 1;

                        while (counter <= 5) {
                            console.log("Counter:", counter);
                            counter++;
                        }

This loop prints numbers from 1 to 5. After each iteration, the value of counter increases until the condition becomes false.

The Do...While Loop

The do...while loop works similarly to the while loop but with one important difference.

The code block executes before the condition is evaluated. This guarantees that the loop will run at least one time.

Syntax

                
                    do {
                        // code to execute
                    } while (condition);

Example

                    
                        let counter = 1;

                        do {
                            console.log("Counter:", counter);
                            counter++;
                        } while (counter <= 5);

Even if the condition eventually becomes false, the code inside the do block will execute first.

Key Differences

Feature While Loop Do...While Loop
Condition check Before execution After execution
Minimum executions 0 1
Common use case When the loop may not run When the loop must run at least once

Example Showing the Difference

                
                    let number = 10;

                    while (number < 5) {
                        console.log("This will never run");
                    }

Because the condition is false initially, the loop does not execute. 

                
                    let number = 10;

                    do {
                        console.log("This runs once");
                    } while (number < 5);

Even though the condition is false, the code executes once because the condition is checked after the first execution.

Conclusion

Both loops are useful when the number of iterations is unknown.

  • Use while when the condition must be validated first.
  • Use do...while when the code must execute at least once.
