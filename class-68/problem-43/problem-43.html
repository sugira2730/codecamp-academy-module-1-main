Programming Challenge

Number Counter using do...while

Problem Overview

In this exercise you will build a simple program that counts how many numbers a user enters. The program must continue asking for numbers until the user decides to stop.

This problem is designed to help you practice the do...while loop, which guarantees that the code inside the loop runs at least once before checking the condition.

Scenario

  • The program asks the user to enter a number.
  • Each time a number is entered, a counter increases.
  • After each entry, the user is asked if they want to continue.
  • If the user answers "yes", the program repeats.
  • If the user answers "no", the program stops.
  • At the end, the program displays how many numbers were entered.

Your Tasks

1. Analyze the Problem

Identify the following elements:

  • Inputs
  • Outputs
  • Variables needed
  • Loop termination condition

2. Design the Algorithm

Create a flowchart that represents the logic of the program.

  • Start and End nodes
  • User input
  • Counter update
  • Decision to continue
  • Loop structure

3. Implement the Solution

Your JavaScript program must:

  • Use a do...while loop
  • Use prompt() to ask the user for numbers
  • Use alert() to display the result
  • Count how many numbers were entered

Example Interaction

                Enter a number: 5
                Do you want to enter another number? (yes/no)

                Enter a number: 10
                Do you want to enter another number? (yes/no)

                Enter a number: 7
                Do you want to enter another number? (yes/no)

                Result:
                You entered 3 numbers.
