Programming Challenge

Password Verification using do...while

Problem Overview

In this challenge you will build a simple password verification program. The program will repeatedly ask the user to enter a password until the correct password is entered.

This exercise is designed to practice the do...while loop, which guarantees that the code inside the loop runs at least once before checking the condition.

Scenario

  • The system has a predefined password stored in the program.
  • The user must enter a password using prompt().
  • If the password is incorrect, the system asks the user to try again.
  • The process repeats until the correct password is entered.
  • When the password is correct, the program displays an access granted message.

Your Tasks

1. Analyze the Problem

  • Identify the inputs
  • Identify the outputs
  • Determine the variables required
  • Identify the loop termination condition

2. Design the Algorithm

Create a flowchart representing the program logic.

  • Start and End nodes
  • User input
  • Password comparison
  • Decision structure
  • Loop repetition

3. Implement the Solution

  • Use a do...while loop
  • Ask for the password using prompt()
  • Compare the password with the correct one
  • Display messages using alert()
