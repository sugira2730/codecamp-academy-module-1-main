Problem 3


Description

Write a program that asks the user to enter a number. The program should determine whether the number is positive or negative or zero.


Conditions

If The number is greater than 0 → it is positive


If The number is less than 0 → it is negative


If The number is equal to zero → it is zero


Output


Display: "The number is positive"


Display: "You number is negative"


Display: "The number is zero"
