# Problem: Find the Largest Number

A program needs to determine the **largest number** in a sequence of values.

In this exercise, the numbers from **1 to 10** will be generated using a loop.

Your task is to write a **JavaScript program using Node.js** that finds the **largest number between 1 and 10**.

## Requirements

- Use a **`for` loop** to iterate from **1 to 10**.
- Use a variable called `largest` to store the **largest number found so far**.
- In each iteration, compare the current number with the value stored in `largest`.
- If the current number is greater, update `largest`.
- After the loop finishes, print the largest number using `console.log()`.

## Expected Output

```
Largest number: 10
```

## Notes

- Do **not** use user input (`prompt`, `readline`, etc.).
- Define all values directly in the code.
- The goal of this exercise is to practice:
  - `for` loops
  - comparison operators
  - tracking a **maximum value**
