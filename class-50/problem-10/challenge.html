🎬 Algorithm Exercise

Movie Theater Ticket System

Design an algorithm that calculates the final price of movie tickets based on several conditions.

Base Ticket Price

The normal price is $12 per ticket.

Age Discounts

  • Under 12 → $6
  • 65 or older → $7
  • 12 – 64 → $12

Student Discount

If the person is a student and age is between 12 and 25 → $3 discount per ticket.

Day Discount

If the day is Wednesday → $2 discount per ticket.

Number of Tickets

The user must enter how many tickets they want to buy.

The algorithm must calculate:

Total Price = final_ticket_price × number_of_tickets

Invalid Input

  • Age less than 0
  • Age greater than 120
  • Tickets less than 1

The program must show:

Invalid input.

Your Tasks

  1. Create the algorithm in JavaScript
  2. Create the flowchart
  3. Use nested conditionals
  4. Calculate the final total price
