# Programming Challenge: Classroom Scores Grid

## Problem Introduction

A teacher stores students' test scores in a **two-dimensional array**. Each row represents a **student**, and each column represents a **test score**.

Example data:

```javascript
const scores = [
  [80, 75, 90],
  [60, 85, 70],
  [95, 88, 92]
];
```

---

## Task

Find the **average score for each student** and print the result.

Example output:

```
Student 0 average: 81.67
Student 1 average: 71.67
Student 2 average: 91.67
```

---

## Constraints

* Use **nested loops**
* Access elements using `array[row][column]`
* Round the average to **2 decimal places**

---

## Learning Objective

* Practice iterating through multidimensional arrays
* Perform calculations using nested loops
* Work with basic data aggregation (sum and average)
