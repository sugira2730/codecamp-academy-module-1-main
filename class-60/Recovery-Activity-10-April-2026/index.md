## Problem: Online Course Certification

An online learning platform evaluates whether a student **passes, fails, or earns honors** in a course based on their **final grade**.

Your program will receive the student's grade as a **variable defined inside the code**.

Using **`if-else` conditionals**, determine the student's result according to the following rules:

| Grade | Result |
|------|------|
| 90 – 100 |  Passed with Honors |
| 70 – 89 |  Passed |
| 50 – 69 |  Remedial Exam Required |
| Below 50 | Failed |

## Program Requirements

Create a file called:

index.js


Inside the program define the grade as a variable, for example:

```javascript
const grade = 85;
```

Use if, else if, and else to evaluate the result and display the output with console.log().

Example output:

```
Final Grade: 85
Result: Passed
```

## Restrictions

Your solution must follow these rules:

- Do NOT request input from the terminal
- Do NOT use switch
- Do NOT use loops
- Use only if, else if, and else
- The grade must be defined as a variable in the code
