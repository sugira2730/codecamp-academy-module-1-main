Switch Statement Exercise

Selecting a mathematical operation using a numeric code in JavaScript.

Problem Introduction

In programming, applications often need to choose between several possible actions depending on a value. One way to implement this decision structure is by using the switch statement.

In this exercise, you will build a small program using JavaScript (Node.js) that selects and executes a mathematical operation based on a numeric operation code.

The program already contains predefined variables. Your task is to implement a switch statement that determines which arithmetic operation should be applied.

  • 1 → Addition
  • 2 → Subtraction
  • 3 → Multiplication
  • 4 → Division

If the operation code does not match any valid option, the program should display an error message using the default case.
