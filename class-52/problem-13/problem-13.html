✈️ Algorithm Exercise

Airport Luggage Fee Calculator

Create an algorithm that calculates the luggage fee for passengers at airport check-in. The final fee depends on passenger type, travel class, and luggage weight.

Passenger Type

  • adult
  • child

If another value is entered:

Invalid passenger type.

Travel Class

  • economy
  • business

If another value is entered:

Invalid travel class.

Luggage Weight

The user must enter the luggage weight in kilograms.

  • Weight must be greater than 0
  • Weight must not exceed 60 kg

If invalid:

Invalid luggage weight.

Free Luggage Allowance

  • Economy: up to 20 kg free
  • Business: up to 30 kg free

Extra Weight Fees

  • Economy: $10 per extra kg
  • Business: $8 per extra kg

Child Discount

If the passenger is a child, the final fee receives a 50% discount.

Your Tasks

  1. Create the algorithm in JavaScript
  2. Use nested conditionals
  3. Validate the inputs
  4. Calculate the luggage fee
  5. Draw the flowchart
