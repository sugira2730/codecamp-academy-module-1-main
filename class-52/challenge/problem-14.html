🎓 Programming Exercise

University Course Grading System

Problem Description

You are developing a grading system for a university course. The system must determine if a student passes, fails, or must take a remedial exam.

The decision depends on:

  • Final grade
  • Attendance percentage
  • Final project submission

📊 Grade Evaluation

Grade Status
90 – 100 Excellent
70 – 89 Pass
50 – 69 Remedial Exam
0 – 49 Fail

📅 Attendance Requirement

Students must have at least 75% attendance to pass the course. If attendance is below this threshold, the student automatically fails.

📁 Final Project Rule

Students with grades between 70 and 89 must submit the final project. Otherwise, they must take a remedial exam.

⚠️ Validation Rules

  • Grade must be between 0 and 100
  • Attendance must be between 0 and 100
  • Project must be yes or no

🧾 Expected Output

                Student grade: 82
                Attendance: 90%
                Final project submitted: yes

                Result: PASS
                Performance level: Good

🎯 Student Tasks

  • Create the algorithm
  • Create the flowchart
  • Implement the solution in JavaScript
