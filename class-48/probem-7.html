Problem 7


Exercise: Bank Loan Approval System


Description

Create a program that determines if a person qualifies for a bank loan. The program must ask the user to enter their monthly salary and their credit score.


Rules

  • The loan is approved only if the monthly salary is at least 2000.
  • The credit score must be 700 or higher.
  • If one or both conditions are not met, the loan must be rejected.

Input

The program must ask the user to enter:

  • Monthly salary
  • Credit score

Example Input

Salary: 2500

Credit Score: 720


Output

The program must display one of the following messages:

  • Loan Approved
  • Loan Rejected
