JavaScript Switch Example

Matching Numbers (1–7) with Days of the Week

Problem Description

In many programs we need to compare a variable against several possible values. When a variable has a limited set of possible values, the switch statement can provide a cleaner and more readable alternative to multiple if / else if conditions.

In this example, we want to create a small algorithm that receives a number between 1 and 7 and prints the corresponding day of the week.

Switch Algorithm

                
                    // Author: Cristian Camilo Cortes Ortiz

                    const dayNumber = 3;

                    switch (dayNumber) {

                    case 1:
                        console.log("Monday");
                        break;

                    case 2:
                        console.log("Tuesday");
                        break;

                    case 3:
                        console.log("Wednesday");
                        break;

                    case 4:
                        console.log("Thursday");
                        break;

                    case 5:
                        console.log("Friday");
                        break;

                    case 6:
                        console.log("Saturday");
                        break;

                    case 7:
                        console.log("Sunday");
                        break;

                    default:
                        console.log("Invalid number");

                }

Explanation

  • The variable dayNumber contains the value we want to evaluate.
  • The switch statement compares this value with each case.
  • When a case matches, the corresponding code block is executed.
  • The break statement stops execution to prevent the program from continuing into the next cases.
  • The default clause runs when none of the cases match the value.

Example Outputs

Input Output
1 Monday
3 Wednesday
