🏦 Algorithm Exercise

Bank Loan Approval System

Design an algorithm that determines whether a bank customer qualifies for a loan. If the loan is approved, the program must calculate the maximum loan amount.

Basic Requirements

  • Age must be between 18 and 65
  • Monthly salary must be greater than 0

If these conditions are not met, the program should show:

Invalid applicant data.

Employment Status

The applicant must enter one of the following:

  • employed
  • self-employed
  • unemployed

If another value is entered:

Invalid employment status.

Credit Score Categories

  • 750 or higher → 5 × salary
  • 650 – 749 → 3 × salary
  • 550 – 649 → 2 × salary
  • Below 550 → Loan rejected

Self-Employed Rule

If the applicant is self-employed, the loan amount must be reduced by 20%.

Unemployed Rule

If the applicant is unemployed:

Loan rejected.

Your Tasks

  1. Create the algorithm using JavaScript
  2. Design the flowchart
  3. Use nested conditionals
  4. Validate all inputs
  5. Calculate the maximum loan amount
