# Challenge: Managing a Shopping Cart with Arrays

## Introduction

In this challenge, you will simulate a **basic shopping cart system** using
JavaScript arrays.  
Arrays are one of the most important data structures in programming because they 
allow us to store and manipulate multiple values in a single variable.

In real applications such as online stores, arrays are often used to store lists 
of items selected by a user. In this exercise, you will practice manipulating an 
array that represents the products inside a shopping cart.

You will apply several **fundamental array operations**, including:

- Checking how many items exist in the array using `length`
- Adding new products to the cart using `push()`
- Removing the last product using `pop()`
- Accessing elements using their index
- Iterating through the array to display all products

Understanding these operations will help you build the foundation for working 
with data collections in JavaScript.

---

## Problem Description

A shopping cart initially contains the following products:

```javascript
let cart = ["Milk", "Bread", "Eggs", "Cheese"];
```

### Write a program that performs the following operations:

1. Print the total number of products currently in the cart.
2. Add a new product "Apples" to the cart.
3. Remove the last product from the cart.
4. Print the first product in the cart.
5. Print the last product in the cart.
6. Display all products in the cart using a loop.
