Online Game Access

An online game wants a program that determines whether a player can enter a special game room depending on their player level and whether they have a VIP membership.

Rules

  • If the player's level is 10 or higher, access is granted.
  • If the level is below 10, the player can still enter if they are VIP.
  • If the level is below 10 and the player is not VIP, access is denied.
  • If the level entered is less than 1, the program must display:
Invalid player level.

Inputs

  • Player level
  • VIP membership (yes / no)

Your Tasks

  1. Create the algorithm using JavaScript.
  2. Use nested conditionals.
  3. Draw the flowchart of the algorithm.
  4. Test the program with different inputs.
