# 🎮 Programming Exercise: Online Game Access

## Problem Description

An online game wants a program that determines whether a player can **enter a special game room** depending on their **player level** and whether they have a **VIP membership**.

---

## Rules

1. If the player's **level is 10 or higher**, the player can **enter the room**.

2. If the player's level is **below 10**, the player can still enter **only if they have VIP membership**.

3. If the player's level is **below 10 and they are not VIP**, they **cannot enter the room**.

4. If the level entered is **less than 1**, the program must display:

> Invalid player level.

---

## Inputs

Your program must ask the user for:

- The **player level**
- Whether the player has **VIP membership** (`yes` or `no`)

Example:

Enter player level:
7

Is the player VIP? (yes/no)
yes

---

## Expected Output Example

Access granted.
