🍔 Programming Exercise

Food Delivery Order

A restaurant wants to create a small program that calculates the final cost of a food order depending on the meal type and whether the customer wants home delivery.

Meal Prices

Meal Price
Burger $8
Pizza $10
Salad $7

Additional Rules

  • If the customer selects home delivery, a $3 delivery fee is added.
  • If the meal is pizza and the customer chooses delivery, the delivery fee becomes $2 (special promotion).
  • If the meal entered is not valid, the program must display:
Invalid meal option.

Inputs

  • Meal type (burger, pizza, salad)
  • Delivery (yes / no)

Your Tasks

  1. Create the algorithm in JavaScript.
  2. Use nested conditionals.
  3. Draw the flowchart of the algorithm.
  4. Test the program with different values.
