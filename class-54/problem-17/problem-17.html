Gym Membership Access

A gym wants to create a program that calculates the membership cost depending on the membership type and whether the user wants personal trainer access.

Membership Types

Membership Price
Basic $20
Premium $35

Additional Rules

  • If the membership is premium, the personal trainer is free.
  • If the membership is basic and the user wants a trainer, add $10.
  • If the membership entered is invalid, the program must display:
Invalid membership type.

Inputs

  • Membership type (basic or premium)
  • Personal trainer access (yes or no)

Your Tasks

  1. Create the algorithm using JavaScript.
  2. Use nested conditionals.
  3. Draw the flowchart of the algorithm.
  4. Test your program with different values.
