# Problem: Sum of Even Numbers

In mathematics, **even numbers** are numbers that can be divided by **2 without a remainder**.

Your task is to write a **JavaScript program using Node.js** that calculates the **sum of all even numbers from 1 to 10**.

## Requirements

- Use a **`for` loop** to iterate from **1 to 10**.
- In each iteration, determine if the number is **even**.
- If the number is even, add it to a variable that stores the total sum.
- At the end of the program, print the final result using `console.log()`.

## Expected Output

```
Sum of even numbers: 30
```

## Notes

- Do **not** use user input (`prompt`, `readline`, etc.).
- Define all values **directly in the code**.
- The purpose of this exercise is to practice **`for` loops, conditional statements, and accumulation**.
