# Problem: Total Points Scored in a Game

In a small video game, a player earns points after completing each level.

The player completed **5 levels**, and the points earned in each level follow 
this pattern:

- Level 1 → 10 points  
- Level 2 → 20 points  
- Level 3 → 30 points  
- Level 4 → 40 points  
- Level 5 → 50 points  

Your task is to write a **JavaScript program using Node.js** that calculates the **total points earned by the player**.

## Requirements

- Use a **`for` loop** to simulate the levels from **1 to 5**.
- In each iteration, calculate the points earned in that level.
- Accumulate the total points in a variable.
- Print the final result using `console.log()`.

## Expected Output

```
Total points earned: 150
```

## Notes

- Do **not** use user input (`prompt`, `readline`, etc.).
- Define all values **directly in the code**.
- The goal of this exercise is to practice **`for` loop iteration and accumulation**.
